Chains

Disc Golf · PWA · Live

A disc-golf caddy that lives in your pocket. Build multiple bags, pull from a library of ~6,300 courses on a GPS map, and walk a round hole-by-hole with a per-shot stat wizard (disc → shot type → landing → execution → result → distance → position → depth → 3×3 basket grid). Tap-in shortcut inside 11ft, aim-point editor, TechDisc CSV import for spin / speed / wobble profiles, UDisc CSV import for historical rounds, JSON export for backup. Course detail pages show layouts, per-hole averages, score-distribution pips, and a best-at-course badge. Installs as a PWA, works offline, and survives mid-round signal drops via Supabase outbox sync. An opt-in AI caddy (Anthropic-proxied via a Cloudflare Worker) recommends discs and lines, with per-user spend caps.

Here There Be Hordes

Game · RTS · Godot

A hand-painted grim-dark survival RTS in true 2:1 isometric — pitched as an industrial nightmare in oil paint: They Are Billions's gameplay with Risen Kingdom's brush. Build and defend a small industrial-era colony against waves of the infected dead. Set up mines, lumberyards, farms, refineries, and munitions workshops; ring them with walls and towers; route survivors with A* pathfinding on a 64×32 tile grid. Every weapon and every building emits noise that pulls more enemies in — a quiet base lasts longer than a loud one. A Terror system tracks how badly things are going for the colony and reacts. Engine: Godot 4.6.2, GDScript. Currently pre-alpha: M0 (Foundation) closed; M1 (Core Loop Vertical Slice) and M2 (Noise System) in flight.

Scheduler

Web App · Cloudflare · v1

An employee-scheduling web app, built first for an Aquatics department (Lifeguard + Swim Instructor sub-departments) and architected to grow into more departments without code changes. Employees sign in with a magic link and set Preferred / Available / Unavailable on a rolling six-week shift window; an auto-fill round-robins through the roster, honoring preferences first, then availability, then min-hours. Managers run a review queue for change requests, shift swaps, time-off, and an open-shift board; admins manage users, org tree, system settings, and an append-only audit log. Mobile-first. Tech: Cloudflare Workers + D1 SQLite for the API, Vite + React + Tailwind on Pages, Supabase Auth for magic-link sign-in.
