Chains

Disc Golf · PWA

A disc-golf caddy that lives in your pocket. Build multiple bags, pull from ~6,300 courses on a GPS map, and walk a round hole-by-hole with a per-shot stat wizard. Tap-in shortcut inside 11ft, aim-point editor, TechDisc CSV import, UDisc import, JSON export. Course detail pages show layouts, per-hole averages, score-distribution pips, and a best-at-course badge. Installs as a PWA, works offline, survives mid-round signal drops via Supabase outbox sync. An opt-in AI caddy (Anthropic via a Cloudflare Worker) recommends discs and lines, with per-user spend caps.

Here Be Hordes

Game · RTS · Godot

A hand-painted grim-dark survival RTS in true 2:1 isometric — pitched as an industrial nightmare in oil paint: They Are Billions's gameplay with Risen Kingdom's brush. Build and defend a small industrial-era colony against waves of the infected dead. Mines, lumberyards, farms, refineries, munitions workshops; walls and towers; A* pathfinding on a 64×32 tile grid. Every weapon and every building emits noise that pulls more enemies in — a quiet base lasts longer than a loud one. A Terror system tracks how badly things are going and reacts. Engine: Godot 4.6.2, GDScript. Pre-alpha: M0 closed, M1 + M2 in flight.

Scheduler

Web App · Cloudflare

An employee-scheduling web app, built first for an Aquatics department (Lifeguard + Swim Instructor sub-departments) and architected to grow into more departments without code changes. Employees sign in with a magic link and set Preferred / Available / Unavailable on a rolling six-week shift window; an auto-fill round-robins through the roster, honoring preferences first, then availability, then min-hours. Managers run a review queue for change requests, shift swaps, time-off, and an open-shift board; admins manage users, org tree, settings, and an append-only audit log. Mobile-first. Tech: Cloudflare Workers + D1 SQLite, Vite + React + Tailwind on Pages, Supabase magic-link.

Launching v1.0 project →

Apothecary Labels

Web Tool · Vanilla JS

A browser-based label designer for Celtic-styled apothecary jars. Type a herb name, get an auto-filled label: parchment background, gold trim, Elder Futhark runes, Celtic symbols, hand-drawn botanical illustrations. Five label sizes from 1.5×0.75″ to 6×3″, all sharing proportions exactly. Optional back label with full description, historic uses, notes, and pairings — front and back print on a single 8.5×11 sheet. Twenty herbs in the lookup with curated Celtic associations, six symbols, the full twenty-four-rune Elder Futhark, custom accent colors. Saves to localStorage. Schema-driven so adding a new format (tea, soap, talisman tag) is a data change, not a rewrite. Vanilla JS, native ES modules, no framework.

Live v0.3.0 launch →

Budget

Local-first · SQLite

A personal budget tool that runs entirely on your own machine. Drop in monthly bank statements, get rule-based categorization across essentials / discretionary / income / transfers, and see where the money is actually going. Dashboard with cash-flow trend, spend-by-category with month-over-month delta, top merchants, essential-vs-discretionary split. Insights surface a subscription detector (3+ similar amounts at monthly cadence), a recurring-spend ranker by annualized cost, and a candidate-cuts list. Bank data never leaves the machine: no cloud, no Plaid, no third-party API touching raw transactions. Vite + React + Tailwind on the front, Hono + better-sqlite3 on the back, pnpm workspace, one importer file per bank format.
