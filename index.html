Here Be Hordes

Game · RTS · Godot

A hand-painted grim-dark survival RTS in true 2:1 isometric. Command a portal-anchored colony on a distant exoplanet, extracting resources for a depleted Earth while holding the native undead at bay. An industrial nightmare in oil paint. Mines, lumberyards, farms, refineries, munitions workshops; walls and towers; A* pathfinding on a 64x32 tile grid. Every weapon and every building emits noise that pulls more enemies in. A quiet base lasts longer than a loud one. A Terror system tracks how badly things are going and reacts. Engine: Godot 4.6.2, GDScript.

Brackish Rising

Game · RTS · Godot

A grim-dark survival RTS in true 2:1 isometric. The deep ocean climbed out of the trenches a hundred and sixty years ago and never went home. Command a fortified Lantern outpost on the inland edge, holding the wall against echo-hunting parasites that wear what is left of the missing. Industrial-medieval and WWI-naval tech ceiling: anything that hums draws them in. Sonic Pressure is the central mechanic; quiet colonies live and loud ones die. Smaller groups of high-HP enemies and longer per-engagement times. Each unit lost is a real loss. Coral encroaches inland year by year. Optional sorties push the perimeter back; survival is the only victory. Engine: Godot 4.6.

Chains

Disc Golf · PWA

A disc-golf caddy that lives in your pocket. Build multiple bags, pull from ~6,300 courses on a GPS map, and walk a round hole-by-hole with a per-shot stat wizard. Tap-in shortcut inside 11ft, aim-point editor, TechDisc CSV import, UDisc import, JSON export. Course detail pages show layouts, per-hole averages, score-distribution pips, and a best-at-course badge. Installs as a PWA, works offline, survives mid-round signal drops via Supabase outbox sync. An opt-in AI caddy (Anthropic via a Cloudflare Worker) recommends discs and lines, with per-user spend caps.

Scheduler

Multi-tenant · Cloudflare

A multi-tenant employee-scheduling SaaS. Each client company gets its own isolated organization on the same instance — cross-org access returns 404, never 403, so tenant existence never leaks. Employees sign in with a magic link and set Preferred / Available / Unavailable on a rolling six-week shift window; an auto-fill round-robins through the roster, honoring preferences first, then availability, then min-hours. Managers run a review queue for change requests, swaps, time-off, and an open-shift board; admins manage users, org tree, settings, and an append-only audit log; admins can own multiple orgs and switch via a header dropdown. A super-admin role observes every org and invites new admins. One-click shareable invite links pre-fill the recipient's login. Mobile-first — verified at 360px. Tech: Cloudflare Workers + D1 SQLite, Vite + React + Tailwind on Pages, Supabase magic-link with JWT verification.

Live v0.2.0 — multi-tenant project →

Apothecary Labels

Web Tool · Vanilla JS

A browser-based label designer for Celtic-styled apothecary jars. Type a herb name, get an auto-filled label: parchment background, gold trim, Elder Futhark runes, Celtic symbols, line-art botanical illustrations from a built-in SpiceJungle library. Five label sizes from 1.5×0.75″ to 6×3″, all sharing proportions exactly. Optional back label with full description, historic uses, notes, and pairings — front and back print on a single 8.5×11 sheet. Nearly two hundred herbs in the lookup with curated Celtic associations, six symbols, the full twenty-four-rune Elder Futhark, custom accent colors, per-instance color + glow overrides on every styled element. Saves to localStorage. Schema-driven so adding a new format (tea, soap, talisman tag) is a data change, not a rewrite. Vanilla JS, native ES modules, no framework.

Budget

Local-first · SQLite

A personal budget tool that runs entirely on your own machine. Drop in monthly bank statements, get rule-based categorization across essentials, discretionary, income, and transfers, and see where the money is actually going. Dashboard with cash-flow trend, spend-by-category with month-over-month delta, top merchants, essential-vs-discretionary split, largest charges, day-of-week heatmap, money-left-this-month tile, forecast. Insights surface a subscription detector (3+ similar amounts at monthly cadence), a per-category MAD outlier classifier, a recurring-spend ranker by annualized cost, a candidate-cuts list, “already cut” and “won't cut” flags with a running money-saved hero. Budgets, account balances, net worth, streaks, calendar heatmap, reconcile, CSV export, and a Cmd-K palette round out the surface. Bank data never leaves the machine: no cloud, no Plaid, no third-party API touching raw transactions; an optional opt-in LLM fallback for categorization sees only normalized merchant strings. Vite + React + Tailwind (OKLCH palette, variable fonts, mesh hero, glass surfaces) on the front, Hono on Node with better-sqlite3 on the back, single pnpm workspace, one importer file per bank format.
