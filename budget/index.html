Quickest path: double-click scripts\yab.cmd from the install directory. That boots both the api and the web app and opens the browser.

From scratch:

  1. Clone github.com/TheProphetKane/yesandbudget
  2. Run pnpm install
  3. Run pnpm dev (or double-click scripts\yab.cmd)
  4. Come back here and click recheck.
