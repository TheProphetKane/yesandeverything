Scheduler

incubating v0.1.0 — design internal tool

An employee-scheduling web app. Employees submit preferred / available / unavailable shifts on a rolling six-week window; an auto-fill rotates through staff, handing out preferred shifts first, then filling gaps from availability. Built first for an Aquatics department (lifeguard + swim instructor), architected to grow into other departments without code changes.

Stack

Cloudflare Pages + Workers + D1. React 18 + TypeScript + Vite + Tailwind on the frontend, Hono on the Worker. Bcrypt password hashing, session cookies, no third-party identity provider.
